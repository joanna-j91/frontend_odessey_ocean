# Ocean Depths

A brief description:
Ocean Depths is an immersive experience that takes users on a vertical journey through the five major zones of the ocean: all the way from the sunlit surface to the darkness of the Hadal trench. 

Built entirely with HTML, CSS, and vanilla JavaScript, the project transforms the deep sea into an interactive narrative that users may enjoy exploring.

The experience opens at the ocean surface animated light rays before descending through five distinct zones, each with its own identity, color palette, and animations. 

The Sunlight Zone features a school of SVG fish that follow the user's cursor in real time. 

The Twilight Zone introduces siphonophores that drift across the section using layered sine wave mathematics. 

The Midnight Zone immerses users in total darkness, broken only by a bioluminescent cursor trail, and an echolocation mechanic that briefly reveals hidden jellyfish and fish schools. 

The Hadal Zone features an amphipod swarm that scatters from the cursor using flocking separation and cohesion forces, chemosynthetic bacteria colonies that pulse in and out, and a sonar ping system with Web Audio API-generated sound, where click position determines both pitch and volume.

A live depth counter tracks the user's scroll position from 0 to 11,000 metres, and scroll-triggered reveal animations, parallax background layers, and zone-specific particle systems reinforce the sense of descending into the unknown.
